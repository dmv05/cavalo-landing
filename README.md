# Cavalo.app — Landing Page

Landing page du projet **Cavalo**, la marketplace mobile 100% gratuite pour l'achat et la vente de chevaux.

Site statique HTML/CSS/JS, déploiement immédiat sur Coolify.

## Structure

```
├── index.html   # Page principale
├── style.css    # Styles (responsive, tons chauds V1)
└── script.js    # Formulaire de pré-inscription (localStorage)
```

## Déploiement

Site statique → déployer sur Coolify en Static Site.
